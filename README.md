# SBS 学习资料库

努力汇集各类计算机相关学习材料，皆在于帮助学习者拓展技术视野，精通计算机工程的创造技艺。知识内容所提供到的不仅仅是技术，更是对于思维上的启发。

名称 SBS (Silver Bullet Shooter) 银弹射手，来源于 Frederick P. Brooks, Jr 的文章 "No Silver Bullet – Essence and Accident in Software Engineering"
